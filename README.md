# network-security-vulnerabilities
Analysis of 8 critical network security vulnerabilities (Cisco, OpenSSH, Fortinet, Ivanti, Palo Alto, HTTP/2, Citrix) — IE3032 Network Security group assignment.
# Network Security Assignment 

**Course:** IE3032 – Network Security  
**Program:** B.Sc. (Hons) in Information Technology, Specialized in Cybersecurity  
**Institution:** Sri Lanka Institute of Information Technology

## Overview
This report analyzes eight critical, recently disclosed vulnerabilities affecting 
common networking devices and protocols. Each vulnerability is examined in terms 
of the affected device/protocol, root cause, exploitation method, real-world 
attack examples, business impact, and recommended mitigations.

## Vulnerabilities Covered
1. **CVE-2025-20333** – Cisco ASA/FTD WebVPN Heap Buffer Overflow
2. **CVE-2024-6387** – OpenSSH "regreSSHion" RCE
3. **CVE-2023-20198** – Cisco IOS XE Web UI Privilege Escalation / RCE
4. **CVE-2023-27997** – Fortinet FortiOS/FortiProxy SSL-VPN Heap Overflow ("Xortigate")
5. **CVE-2025-0282** – Ivanti Connect Secure Stack-Based Buffer Overflow
6. **CVE-2024-3400** – Palo Alto PAN-OS GlobalProtect Command Injection
7. **CVE-2023-44487** – HTTP/2 Rapid Reset Attack
8. **CVE-2023-4966** – Citrix NetScaler "Citrix Bleed"
